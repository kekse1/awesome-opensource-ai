# Contributing

Thanks for contributing to Awesome Open Source AI.

## Inclusion criteria

Projects added to this list should be:

- Truly open source with an OSI-approved license
- Actively maintained
- Relevant to the open-source AI ecosystem
- High quality, useful, and reasonably well documented

## Formatting

- Keep entries concise and factual
- Prefer the official project repository or official project website
- Use the existing section structure whenever possible
- Add GitHub star badges for GitHub-hosted projects
- Do not add trailing parenthetical metadata notes at the end of bullets

## Before opening a PR

- Check for duplicates
- Place the project in the best-fit section
- Keep descriptions short and neutral
- Make sure links work

## Pull requests

- Small focused PRs are preferred
- If you add a new category, explain why it is needed
- Be ready to update formatting if maintainers request changes
